# API-JSON

A simple CLI tool that fetches data from the ISRO API and displays structured results.

## Features
- Fetches customer satellite data from ISRO's public API.
- Converts JSON data into a structured format.
- Displays results in a readable manner.

## Prerequisites
- Go (Golang) installed on your system.
- Internet connection to fetch API data.

## Installation
Clone the repository:
```sh
$ git clone https://github.com/anicse37/API-JSON.git
$ cd API-JSON
```

Build the project:
```sh
$ go build -o api-json
```

## Usage
Run the CLI tool:
```sh
$ ./api-json
```
This will fetch the data from ISRO's API and display the structured output.

## API Source
Data is fetched from:
[https://isro.vercel.app/api/customer_satellites](https://isro.vercel.app/api/customer_satellites)
