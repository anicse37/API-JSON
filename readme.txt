This Program is to learn about API and JSON conversion

This program is using an API("https://isro.vercel.app/api/customer_satellites") to get data in JSON format and links it with structure for ease of access. 

It displays the ID, Country, Launch date, and Launcher of the satellites that ISRO has launched into the space 

To Run the Program:-
(in command line)
             -> go run API_JSON_ExampleProgram.go
